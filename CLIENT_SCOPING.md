## 3. Brand & Design

- **Do you have a logo?** (PNG or SVG preferred — we need the file)
- **Do you have existing brand colours?** If yes, list hex codes. If no, describe the vibe you want (e.g. "dark and premium", "clean and bright", "bold and energetic")
- **Do you have preferred fonts?** If not, we'll select from our design system.
- **Any websites you like the look of?** (2-3 reference links help a lot)

---

## 4. Pages & Content

Which pages do you need? (All included by default — we remove what you don't need.)

- [ ] Home
- [ ] About / Founder bio
- [ ] Services
- [ ] Contact
- [ ] Privacy Policy (required for GDPR — we generate this)
- [ ] Blog / Newsletter (links to external platform)

**For each page, we need:**

### Home
- **Hero headline:** (one punchy line — we can help write this)
- **Hero subline:** (1-2 sentences expanding on the headline)
- **Primary CTA text:** (e.g. "Contact Us", "Get Started", "Book a Call")
- **Secondary CTA text:** (e.g. "Our Services", "Learn More")

### About
- **Founder name:**
- **Founder role/title:**
- **Bio:** (2-3 paragraphs — who you are, what drives the work)
- **Company philosophy:** (1-2 sentences — what principle guides everything?)
- **Core values:** (3 values with a one-line description each)

### Services
- **How many services?** (typically 3-5)
- **For each service:**
  - Name
  - Description (2-3 sentences)
- **Process steps:** How do you work with clients? (typically 3-4 steps, e.g. Discovery → Strategy → Build → Handover)

### Contact
- **Contact heading:** (default: "Get in Touch")
- **Response time:** (e.g. "within 24 hours")
- **Preferred form fields:** Name, email, message are default. Do you also want: company name, phone, budget range, service type dropdown?
